## Getting started

1. clone this repository

   ```sh
   git clone https://github.com/TEAM3-ED8/back-end.git
   ```

2. install dependencies

   ```bash
   pnpm install
   ```

3. create .env file

   ```bash
   touch .env
   ```

4. write to .env

   ```bash
   DATABASE_URL="postgresql://postgres:PpVknpeLsATaSpBPvAGPgveCUqtMSDwc@junction.proxy.rlwy.net:29280/railway"
   ```

5. run the project
   ```bash
   pnpm run dev
   ```
